- **Q5. 성장하기 위해 지금 어떤 노력을 하고 있는지 설명해주세요.**
    
    배운게 많이 없기때문에 제가 할수있는 기초지식에서  여러번 반복하고 많은자료들을 찾아보고있습니다!
    
    -한가지에만 몰두하는게 아닌 여러가지에 초점을두고 많은것을 경험하고 사용해보려고 노력하고있습니다.

    - **Q1. 크로스 브라우징이란 무엇이며, 왜 필요한가요?**
    
    크로스 브라우징이란 웹 페이지 제작 시에 모든 브라우저에서 깨지지 않고 의도한 대로 올바르게(호환성) 나오게 하는 작업을 말합니다.
    
    필요한 이유는 브라우저마다 랜더링 되는것이 다르기 떄문입니다. 
    
- **Q2. HTML은 무엇의 약자이고 어떤 걸 의미하는지 또한 HTML 파일에서 <!DOCTYPE html> 이 의미하는 걸 설명해 주세요.**

    DOCTYPE은document type의 약어로, 웹 문서가 어떠한 형식으로 작성되었는지 최상단에 문서의 형식을 선언하는 것을 의미합니다
    
- **Q3. CSS는 무엇의 약자이고 어떤 걸 의미하는지 설명해 주세요.**
    
    CSS란 Cascading Style Sheetsd의 약자로, 문서의 스타일을 꾸밀때 사용합니다
    
- **Q4. 웹 접근성에 대해 설명해주세요.**
    
    웹(Web) 접근성이란 디지털분야에서 웹사이트,도구,기술적장애를 가진 사람들도 이용하는것
    
     고령자나 몸이 불편한사람을 위한 정보를 일반인들과 동등하게 이용할수있도록 보장해주는것입니다.
    
- **Q5.웹 접근성을 잘 구현한 예시로 어떤 게 있을까요?**
    
    시각장애인을 위한 음성인식이나, 눈이 안좋은 사람을위한 화면확대도구 등이 있습니다.
